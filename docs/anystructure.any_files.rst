Legacy module reference
=======================

This page is kept for old links. The current Python import package is
``anystruct``; the old ``anystructure.any_files`` module path is no longer the
public API. Use :doc:`api`, :doc:`api_examples`, and :doc:`anystructure` for the
current API and source-module documentation.

Current import examples:

.. code-block:: python

   from anystruct.api import FlatStru, CylStru
   from anystruct import calc_structure, calc_loads
