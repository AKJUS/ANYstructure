External PULS package
=====================

The old external spreadsheet-driven DNV PULS workflow has been removed from
the current release. Flat plate buckling alternatives available through the
public API are:

* ``DNV-RP-C201 - prescriptive``
* ``SemiAnalytical S3/U3``
* ``ML-Numeric (PULS based)``

See :doc:`api` and :doc:`api_examples` for current usage.
